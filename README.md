# Efficient ML Systems Final Project
## LoRA vs Full Fine-Tuning vs Head-Only

### 📌 Overview
This project analyzes the efficiency-performance trade-offs of:
- Full Fine-Tuning
- LoRA (Low-Rank Adaptation)
- Head-Only Training

on DistilBERT for text classification.
